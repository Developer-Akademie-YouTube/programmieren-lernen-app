JS Basics

Einfach & mobil lernen

1. Console.log

Bitte schaue dir den nachfolgenden Code an:
console.log('Hallo Welt!');
Übung: Was macht dieser Code

2. Variablen

Speichere Daten mit let:
let myName = 'Junus';
console.log('Hallo ' + Junus);
Übung: Was wird angezeigt?

3. String-Konkatenation

Verknüpfe mit +:
console.log("JS" + " Rocks");
Übung: Was zeigt die Konsole?

4. Kommentare

Kommentare mit // oder /* ... */:
// console.log('Das wird nicht ausgegeben');
/* console.log('Auch nicht'); */
console.log('Nur dieser Text wird ausgegeben');
Übung: Was erscheint in der Konsole?

5. Zahlen und Rechnen

Einfaches Rechnen:
console.log(2 + 3 * 4);
Übung: Was zeigt die Konsole?

6. Vergleichsoperatoren

Gleichheit prüfen:
console.log(5 == '5');
Übung: Was wird angezeigt?

7. Strikte Gleichheit

Typ und Wert prüfen:
console.log(5 === '5');
Übung: Was zeigt die Konsole?

8. if-Anweisung

Bedingt ausführen:
let x = 10;
if (x > 5) {
  console.log('Groß');
} else {
  console.log('Klein');
}
Übung: Was erscheint?

9. for-Schleife

Schleife von 0 bis 2:
for (let i = 0; i < 3; i++) {
  console.log(i);
}
Übung: Welche Zahlen werden ausgegeben?

10. while-Schleife

Schleife solange i < 2:
let i = 0;
while (i < 2) {
  console.log(i);
  i++;
}
Übung: Was zeigt die Konsole?

11. Funktionen

Einfach definieren und aufrufen:
function sagHallo() {
  console.log('Hallo!');
}
sagHallo();
Übung: Was passiert?

12. Parameter

Mit Parameter arbeiten:
function add(a, b) {
  console.log(a + b);
}
add(3, 4);
Übung: Was zeigt die Konsole?

13. Arrays

Ein Array erstellen:
let fruits = ['Apfel', 'Banane', 'Kirsche'];
console.log(fruits[1]);
Übung: Was wird ausgegeben?

14. Array push

Ein Element hinzufügen:
let arr = [1, 2];
arr.push(3);
console.log(arr.length);
Übung: Was zeigt die Konsole?

15. Objekte

Objekt mit Eigenschaften:
let person = {name: 'Anna', age: 25};
console.log(person.name);
Übung: Was wird ausgegeben?

16. Template-Strings

Mit Backticks und ${}: 
let name = 'Ben';
console.log(`Hallo, ${name}!`);
Übung: Was zeigt die Konsole?

17. Arrow-Funktionen

Kurzschreibweise:
const quad = x => x * x;
console.log(quad(5));
Übung: Was wird ausgegeben?
